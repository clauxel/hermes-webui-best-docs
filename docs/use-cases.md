# Use Cases - Hermes WebUI Launch Lab

Hermes WebUI Launch Lab scans redacted deployment plans and turns install, Docker, WSL2, SSH, gateway, and provider risks into launch receipts.

## Docker launch review

Check a compose-based Hermes WebUI rollout before exposing it to users.

## WSL2 onboarding

Help a Windows operator understand where state, ports, and provider credentials should live.

## SSH access planning

Verify a tunnel or reverse proxy plan without leaking secrets into support channels.

## Team readiness report

Create a launch receipt for platform, support, or security review.

## Pricing and Support

Plans cover readiness scans, redacted evidence parsing, owner notes, shared reports, launch receipts, and export workflows.
