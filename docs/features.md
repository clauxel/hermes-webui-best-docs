# Features - Hermes WebUI Launch Lab

Hermes WebUI Launch Lab scans redacted deployment plans and turns install, Docker, WSL2, SSH, gateway, and provider risks into launch receipts.

## Install readiness scan

Review host, port, state directory, password, workspace mount, provider setup, and expected launch path.

## Docker deployment audit

Check compose files, volumes, environment variables, reverse proxy assumptions, and persistent state.

## Windows and WSL2 review

Separate Windows shell, WSL2 filesystem, Docker Desktop, and network assumptions before launch.

## SSH tunnel safety check

Inspect tunnel direction, exposed ports, authentication, and operator access before a browser surface is shared.

## Redacted evidence reports

Turn copied setup notes into a report that removes secrets and gives owners clear next actions.

## Team launch receipts

Unlock shared reports, exportable receipts, issue owners, and recurring launch history after checkout.

## Pricing and Support

Plans cover readiness scans, redacted evidence parsing, owner notes, shared reports, launch receipts, and export workflows.
