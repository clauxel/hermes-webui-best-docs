# How It Works - Hermes WebUI Launch Lab

Hermes WebUI Launch Lab scans redacted deployment plans and turns install, Docker, WSL2, SSH, gateway, and provider risks into launch receipts.

## Paste a redacted plan

Add install notes, compose snippets, tunnel plan, host choice, or onboarding status without secrets.

## Scan launch blockers

Review provider, gateway, workspace, persistence, auth, and exposure risks.

## Assign next actions

Turn each blocker into an owner note, verification step, or safe fallback.

## Export launch receipt

Use a paid Team report when the deployment needs a shareable record.
